# Ruby CLI Calculator

A simple command-line calculator built with Ruby. It allows users to input two numbers and an arithmetic operator to perform calculations.

## Features

- Supports basic arithmetic operations (+, -, *, /)
- Accepts integers and decimals (comma or dot)
- Runs continuously until the user exits
- Handles invalid operations gracefully

## Usage

1. Clone the repository or download the file
2. Run the script:

```bash
ruby calculator.rb
```

3. Follow the prompts in the terminal
4. Type "exit" (or "sair") to quit

## Example

Enter the first number: 10

Enter the operator: +

Enter the second number: 5

Result: 15
